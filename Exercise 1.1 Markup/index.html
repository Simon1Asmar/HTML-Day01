Quotation Element

Details & Summary

Summary Element (click to open)

the details element is for additional details that the user can open and close if they're interested in knowing more. The Summary element is the element inside the details element that shows the visible part that the user can click on to view the details

Only one summary element can be inside the details element.

Code Element

The code element is used to specify that the written text is actually a code. By default it uses the monospace font. Here's an example: int x = 1;

Cite Element

the cite element is used to when talking about a title of a creative work like a book, tv show, movie, art piece...
Ex.
The 48 Laws of Power by Robert Greene.
